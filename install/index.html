Install Ambulance App

This app is designed to run as a secure web clip on your iPhone or iPad. Please install it once so you can access all protected tools like Ambulance and CPR.

⬇️ Download Configuration Profile

1. Download the Profile

Tap the button above. Safari will say “Profile Downloaded”.

2. Open Settings

Go to Settings → Profile Downloaded at the top.

3. Install the Profile

Tap Install, enter passcode if asked, then tap Install again to confirm.

4. Done

Your Ambulance app will now appear on your Home Screen.
